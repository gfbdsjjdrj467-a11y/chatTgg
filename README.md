# ChatTgg — Web demo

This is a small static web demo inspired by Telegram Web styles (buttons and animations). It is intended as a clean, lightweight landing/chat UI that you can publish with GitHub Pages.

How to enable on GitHub Pages
1. Push these files to the `main` branch of the repository `gfbdsjjdrj467-a11y/chatTgg` (they are already prepared to live at the repo root).
2. In your repository settings → Pages, set Source to `main` branch / root and save.
3. After a minute, your site will be available at `https://gfbdsjjdrj467-a11y.github.io/chatTgg/` (or the Pages URL GitHub provides).

What I added
- index.html — minimal single-page chat-like UI
- styles.css — Telegram-ish color theme, button styles, ripple/pulse animation
- script.js — minimal client JS to send messages and fake replies
